2.3 Exercice : Secrets dans Docker Compose (20 min)

1. Ajoute un service web qui se connecte à la base de données en utilisant le secret :
   - Dans le même répertoire, modifie le fichier docker-compose.yml pour inclure un service web (ex. un conteneur Nginx) :
     version: '3.7'
     services:
       postgres:
         image: postgres
         secrets:
           - db_password
         environment:
           POSTGRES_PASSWORD_FILE: /run/secrets/db_password

       web:
         image: nginx
         ports:
           - "8080:80"
         depends_on:
           - postgres

     secrets:
       db_password:
         external: true

2. Lance le service :
   docker-compose up -d
